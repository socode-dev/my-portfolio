# SOCODE Portfolio

A modern, responsive front-end developer portfolio built with:

- ⚛️ React (Vite)
- 🎨 TailwindCSS
- 🌀 Framer Motion
- 🧭 React Router DOM

## 🚧 Status

**Phase 1 complete** — project initialized and environment configured.

## 📦 Getting Started

1. Clone this repo
2. Run `npm install`
3. Start dev server: `npm run dev`
